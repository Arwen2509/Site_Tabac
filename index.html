Infos pratiques pour le retrait

J'ai un QR code

Si votre transporteur fournit un QR code valide pour le retrait, il peut suffire pour recuperer votre colis.

Je n'ai pas de QR code

Dans ce cas, une piece d'identite reste generalement necessaire pour verifier le retrait.

Je retire pour quelqu'un d'autre

Il vaut mieux appeler avant de venir, car les conditions peuvent varier selon le transporteur.

Je n'ai pas recu de message

Si vous n'avez pas encore recu le SMS ou l'email du transporteur, votre colis n'est peut-etre pas encore disponible au retrait.
